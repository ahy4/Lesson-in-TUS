    - 因子分析など

4. 多次元尺度解析(MDS)


## 計測尺度

* 原点を選ぶことに意味がある

* 比に意味

* (身長の統計など)

## 間隔尺度

* げんてんは便宜的

* 差の比に意味

* (駅たちの距離など)

順序尺度

* 大きさの順序のみ意味

* (レースの順位、タイムなど)

## 名義尺度

* 違うものはちがう

## 定義4.2

<img src="https://latex.codecogs.com/gif.latex?%5Cdpi%7B120%7D%20K%5BX%5D">とIはK上の線形空間であるので(IはKの部分空間)

K[X]/Iは(乗算を無視して)K上の線型空間としての商空間ともみなせる

これの線形空間としての次元をdim K[X]/I で表す

### 例

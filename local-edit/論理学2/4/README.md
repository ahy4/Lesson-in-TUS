## ASの完全性

### 弱い完全性：

任意の式αについて

M|=α<=>AS|-α

### 強い完全性

AS-|α (αのASに対する独立性)

AS-|αかつAS-|~α

であるような式αは存在しない

SPC(標準命題論理(?))の体型において「αが直感的に真である」

ということを「αはSPC恒真式である」といい、

SPC|=α

とかく。その定義として

SPC|=α <=(def)=>  任意の付値関数$V_i$について$V_i[\alpha]=1$

SPC=|αと書く時、これを「αはSPC恒偽式である」

とよみ、

SPC=|α <=(def)=> 任意の付値関数$V_i$について$V_i[\alpha]=0$


SPCを公理化したPM公理化について、

任意の式αについて、「αがPM証明可能式である」

PM-|α

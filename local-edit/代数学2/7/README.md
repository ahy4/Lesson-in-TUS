## Claim 1

$$f'(X_1,...,X_{n+1})=f(Y_1+a_1Y_{n+1},...,Y_n+a_nY_{n+1},Y_{n+1})\\
=h(a_1,...,a_n)Y_{n+1}^N+t$$

hは0でないn変数多項

fはy_n+1に関して素数N未満

### 例
$$
f(x_1,x_2,x_3)=3x_1x_2x_3+x_1^2+3x_1+x_2+1\\
f(y_1-a_1y_3,y_1-a_2y_3,y_3)\\
=[[3a_1a_2y_3^3+a_1^2a_2y_3^3]]+y_3 未満に関して3未満の式\\
=p(a_1,a_2,a_3)+y_3...
=h(a_1,a_2)
$$

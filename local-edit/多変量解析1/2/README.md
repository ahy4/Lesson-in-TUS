# 多変量解析

n人p変量

$X = (\Bbb x_1, \Bbb x_2, ..., \Bbb x_n),\ \Bbb x_i \in \Bbb R^p$

を下と同一同一視する

$\mathbb x_{\rho (1)}, ..., \mathbb x_{\rho (n)},\ \rho \in S_n$

同一視してうしなわれる情報を対象にする解析

1. 主成分分析

2. 正準相関

    - 重回帰、判別、数量化Ⅰ〜Ⅲ類など

3. 潜在構造分析

    - 因子分析など

4. 多次元尺度解析(MDS)


## 計測尺度

* 原点を選ぶことに意味がある

* 比に意味

* (身長の統計など)

## 間隔尺度

* げんてんは便宜的

* 差の比に意味

* (駅たちの距離など)

順序尺度

* 大きさの順序のみ意味

* (レースの順位、タイムなど)

## 名義尺度

* 違うものはちがう
